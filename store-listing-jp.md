# Chrome Web Store 提出用ドラフト

## ストア掲載情報（日本語）

### 拡張機能名
SplitStream Browser

### 短い説明
2列/3列でページを連続的に読む Chromium 拡張。スクロールに追従して縦方向の流れを分割表示します。

### 長い説明（公開用）
Web を読むとき、長文ページは“1 本の縦ストリーム”で流れ、行長や視線移動に疲れやすいことがあります。SplitStream Browser は、ページを2列（将来3列）に分割し、`左→右→下`の順で“川のように”連続する表示に変換します。

- 縦スクロールをそのまま維持
- 左右の表示が連続し、行の追従が切れにくい
- `s+2`, `s+3` で必要時に切替
- `s+1` で通常表示に戻す

読みにくいと感じる場面でだけ切り替えられるため、作業導線を止めずに読みやすさを最適化できます。

### カテゴリ（申請時選定）
- Accessibility
- Productivity

### スクリーンショット（推奨）
1. 通常表示
2. 2分割（分割境界線あり）
3. 3分割（ヘビ状に接続される様子）
4. スクロール中の見え方

### 1:1での主要機能説明
- 同一ページをページ内で複数カラム化
- 連続スクロール同期待機（1つのマウスホイール/スクロールで全体が動く）
- オン/オフ/分割数の切替

### プライバシー
- 収集するユーザーデータ: なし
- 処理対象: 表示中ページの DOM とスクロール状態のみ
- 外部通信: なし
- 開発者側サーバー通信: なし
