# Snake-Scrollable Split Viewer (Chromium Extension)

## 1. これを作りたい理由

最近の多くの Web サイトはスマホ想定のレイアウトで、PC では見た目が窮屈に感じる一方、横長モニターのメリットを活かしにくい場面があります。

`Snake-Scrollable Split Viewer` は、ページの読みやすさを上げるために「1 つのページを 2 列（将来 3 列）で連続的に流す」ことを目指した拡張機能です。

- 左右の列で見やすい幅に収めて表示
- スクロールしても各列が独立せず連続して流れる（ヘビ状接続）
- 必要なときだけ表示方法を切り替え可能

## 2. 使い方

### インストール（開発版）

1. `chrome://extensions` を開く
2. 右上の**デベロッパーモード**を有効化
3. **パッケージ化されていない拡張機能を読み込む**でこのフォルダを選択
4. 拡張機能を開いて `s` → `1 / 2 / 3` でモード切り替え

### 操作（ショートカット）

- `s` → `1`: オフ（通常表示）
- `s` → `2`: 2 分割（左→右が連続）
- `s` → `3`: 3 分割（実験的）

> 備考: 「Shift+1/2/3」ではなく「`s` キー直後 + 数字キー」です。

## 3. フォルダ構成

- `manifest.json`: 拡張機能定義
- `background.js`: メッセージングとタブ制御
- `content.js`: 分割表示・スクロール同期の本体
- `popup.html` / `popup.js`: デバッグ用UI
- `run-splitstream.ps1`: 補助スクリプト

## 4. 開発検証コマンド

```powershell
node --eval "const fs=require('fs'); JSON.parse(fs.readFileSync('manifest.json','utf8')); console.log('manifest ok')"
```

## 5. ストア掲載向けコピー（ドラフト）

短い説明（72文字前後）:
- 2列/3列で連続スクロール表示する読書支援 Chrome 拡張。画面を分割して見出しや本文を疲れにくくします。

長い説明（ストア掲載用）:
- PC での長文閲覧は、ページは縦に長いのに表示幅が広すぎる、または逆にスマホ向けの短い行が連続して読みにくい、という摩擦が起きがちです。
- `Snake-Scrollable Split Viewer` はページを 2 列（または 3 列）にわけ、左上から右上へ、次に下段へと続けて読む“連続的な縦流れ”を維持します。
- 分割表示のオン/オフや段数はショートカットで即時切替。読みづらい場面でだけ分割を有効化できるため、作業フローを止めずに読み疲れを減らせます。
- 本拡張は既存サイト表示を壊さないことを前提に、同一ページを同期待機のまま複数カラム化する実験的な読み取り支援ツールです。
